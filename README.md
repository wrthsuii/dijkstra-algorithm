# Алгоритм Дейкстри - Багатомовна реалізація

Реалізація класичного алгоритму Дейкстри для пошуку найкоротшого шляху у графі в трьох парадигмах програмування.

## Структура проєкту

```
dijkstra-algorithm/
├── Prolog/
│   ├── dijkstra.pl      # Реалізація на Prolog (345 рядків)
│   └── README.md        # Опис Prolog реалізації
├── Haskell/
│   ├── dijkstra.hs      # Реалізація на Haskell (160 рядків)
│   └── README.md        # Опис Haskell реалізації
├── Python/
│   ├── dijkstra.py      # Реалізація на Python (93 рядки)
│   └── README.md        # Опис Python реалізації
├── COMPARISON.md        # Детальне порівняння реалізацій
└── README.md            # Цей файл
```

## Реалізації

**[Prolog](Prolog/)** - Логічне програмування
- Граф як факти, декларативний опис алгоритму
- Мультипризначенність предикатів, автоматичний backtracking
- 345 рядків коду

**[Python](Python/)** - Імперативне програмування
- Граф як матриця суміжності, клас Graph
- Покрокове виконання з явним контролем стану
- 93 рядки коду

**[Haskell](Haskell/)** - Функційне програмування
- Граф як Map, незмінні структури даних
- Чисті функції, композиція, рекурсія
- 160 рядків коду

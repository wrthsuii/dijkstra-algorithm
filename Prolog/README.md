# Реалізація на Prolog

## Опис

Декларативна реалізація алгоритму Дейкстри засобами логічного програмування. Граф представлено як множину фактів `edge(From, To, Weight)`. Алгоритм описує "що" потрібно знайти через систему правил та предикатів.

**Головний предикат:** `dijkstra(Start, End, Path, Distance)`

## Особливості

- Мультипризначенність: пошук, перевірка, обчислення
- Рекурсивна обробка через предикати
- Автоматичний backtracking
- Незмінні структури даних (списки)

## Приклад

```prolog
?- dijkstra(a, e, Path, Dist).
Path = [a, c, f, e], Dist = 20.

?- dijkstra(a, e, [a,c,f,e], 20).
true.
```

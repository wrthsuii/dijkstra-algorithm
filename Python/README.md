# Реалізація на Python

## Опис

Імперативна реалізація алгоритму Дейкстри з використанням ООП. Граф зберігається як матриця суміжності в класі `Graph`. Покрокове виконання з явним управлінням станом через цикли та умовні конструкції.

**Головний метод:** `dijkstraWithPath(src, dest)`

## Особливості

- Матриця суміжності O(V²)
- Мутабельні структури даних
- Ітеративна обробка (цикли for/while)
- Складність: O(V²)

## Приклад

```python
g = Graph(6)
g.graph = [[0, 7, 9, ...], ...]
find_shortest_path(0, 4)
# Output: Path: 0 -> 2 -> 5 -> 4
#         Total Distance: 20
```

## Запуск

```bash
python djikstra.py
```

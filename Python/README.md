# Реалізація алгоритму Дейкстри на Python
## Опис проєкту
**Постановка задачі:** 

“Найкоротший шлях у графі” – це класична задача теорії графів, яка полягає у знаходженні шляху між двома заданими вершинами графа, для якого сума ваг ребер, з яких він складається, є мінімальною серед усіх можливих шляхів.

**Короткий опис обраного алгоритму:** 

Алгоритм Дейкстри знаходить найкоротші шляхи від однієї вершини до всіх інших у графі з невід’ємними вагами ребер. Він поступово обирає вершину з мінімальною відстанню та оновлює відстані до суміжних вершин до тих пір, поки всі вершини не будуть опрацьовані.

**Використаний підхід:**

Алгоритм Дейкстри реалізовано у імперативному стилі програмування. Реалізація у вигляді класу ``Graph``, де граф зберігається як матриця суміжності. Використовується допоміжна функція ``minDistance`` для пошуку вершини з мінімальною відстанню та функція ``dijkstraWithPath`` для обчислення найкоротшого шляху та відновлення маршруту.

**Висновок:**

Реалізація Дейкстри на Python у імперативному стилі забезпечує простий контроль за обчисленнями та наочне відстеження кроків алгоритму.
